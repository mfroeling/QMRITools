DTItoolExpTens

DTItoolExpTens[tensor]
exports a diffustion tensor array to the DTItool format (*.dat).

DTItoolExpTens[tensor, add]
exports tensor and adds - add to the filenames.

DTItoolExpTens[tensor, add, folder]
exports tensor to the given folder and adds - add to the filenames.
