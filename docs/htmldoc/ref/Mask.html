Mask

Mask[data]
creates a mask by automatically finding a threshold.

Mask[data, min]
creates a mask which selects only data above the min value.

Mask[data,{min,max}]
creates a mask which selects data between the min and max value.

DetailsDetails

  • The following options can be given:
  • MaskSmoothingFalseMaskSmoothing is an options for Mask, if set to True it smooths the mask, by closing holse and smoothing the contours.
    MaskComponents1MaskComponents is an option for Mask and SmoothMask. Determinse the amount of largest clusters used as mask.
    MaskClosing5MaskClosing is an option for Mask and SmoothMask. The size of the holes in the mask that will be closed
    MaskFiltKernel2MaskFiltKernel is an option for Mask, SmoothMask and SmoothSegmentation. How mucht the contours are smoothed.
