TransformData

TransformData[{data,vox}]
deforms the data according to the last output of register data. The directory should be the TempDirectory were the registration is stored. DeleteTempDirectory should be False.

DetailsDetails

  • The following options can be given:
  • TempDirectoryDefaultTempDirectory is an options for RegisterData, RegisterDiffusionData, RegisterCardiacData and RegisterDataTransform. It specifies the temprary directory used to perform and output the registration.
    FindTransformAutoFindTransform is an option for TransformData and RegisterTransformData. It specifies where to find the transformfile.
    DeleteTempDirectoryAllDeleteTempDirectory an options for RegisterData, RegisterDiffusionData, RegisterCardiacData and RegisterDataTransform. It specifies if the temp directory should be deleted after the registration is finisched.
    PrintTempDirectoryTruePrintTempDirectory is an options for RegisterData, RegisterDiffusionData, RegisterCardiacData and RegisterDataTransform. It spefifies if the location of the temp directory should be deplayed.
