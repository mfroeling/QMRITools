AutoCropData

AutoCropData[data]
crops the data by removing all background zeros.

AutoCropData[data,pad]
crops the data by removing all background zeros with padding of pad.

DetailsDetails

  • The following options can be given:
  • CropPadding5CropPadding is an option for AutoCropData or FindCrop. It specifies how much padding to use around the data.
