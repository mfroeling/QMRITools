SumOfSquares

SumOfSquares[{data$1, data$2, . $$ datan}]
calculates the sum of squares of the datasets. Output is the SoS and the weights, or just the SoS.

DetailsDetails

  • The following options can be given:
  • OutputWeightsTrueOutputWeights is an option for SumOfSqares. If True it also output the SoS weights.
