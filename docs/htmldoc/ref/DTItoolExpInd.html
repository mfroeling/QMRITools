DTItoolExpInd

DTItoolExpInd[data, file]
exports a 3D array data to the file filename DTItool format (*.dat) using DatWrite.

DTItoolExpInd[data, file ,folder]
exports data to given file and folder.

DTItoolExpInd[data, file ,folder, add]
exports data to given file and folder and adds -add to the filename.
