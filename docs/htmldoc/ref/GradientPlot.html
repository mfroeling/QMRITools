GradientPlot

GradientPlot[bvec, bval]
plots the given bvec with position of the gradients scaled according to the bval.

DetailsDetails

  • The following options can be given:
  • PlotSpacebspacePlotSpace is an option for GradientPlot can be "bspace" or "qspace".
    PlotColorAutoPlotColor is an option for GradientPlot can be any color or gradient color name.
    SphereSize0.05SphereSize is an option for GradientPlot and ListSpherePlor. Sets the size of the spheres thar represent the gradients.
    PositiveZFalsePositiveZ is an options for GradientPlot. If True all Gradients are displayed with a positive z direction.
