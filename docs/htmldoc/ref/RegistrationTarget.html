RegistrationTarget

RegistrationTarget
is an option for RegisterDiffusionData and RegisterCardiacData. Specifies which target to uses for registration if using "rigid", "affine" or "bspline" as MethodReg. If the MethodReg is "PCA" based it does not need a target and this options does nothing. Values can be "First", "Mean" or "Median".
