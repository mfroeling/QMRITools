CorrectGradients

CorrectGradients[grad, transformation]
corrects the gradient directions grad with the tranformation parameters from RegisterData or RegisterDiffusionData. Output is the corrected gradient vector.

DetailsDetails

  • The following options can be given:
  • MethodRegRotationMethodReg is an options for RegisterData, RegisterDiffusionData, RegisterCardiacData and RegisterDataTransform. It spefifies which registration method to use. Mehtods can be be "translation", "rigid", "affine", "bspline", "rigidDTI", "affineDTI", "PCAtranslation", "PCArigid", "PCAaffine", or "PCAbspline".
