ParameterCalc

ParameterCalc[tensor]
caculates the eigenvalues and MD and FA from the given tensor. The parameters are l1, l2, l3, MD and FA. l1, l2, l3, MD are in (10^-3 mm^2/s)

DetailsDetails

  • The following options can be given:
  • RejectTrueReject is an option for EigenvalCalc. It True then voxels with negative eigenvalues are rejected and set to 0.
    MonitorCalcFalseMonitorCalc is an option for all Calc fucntions. When true the proceses of the calculation is shown.
